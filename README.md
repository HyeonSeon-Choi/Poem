# 다정한 침입 — 최현선 시집

**다정한 침입** · *속수무책의 성실* — 최현선 시집을 위한 정적 웹사이트.
3D로 책장을 넘기는 플립북, 시마다 어울리는 일러스트, 중앙 정렬의 감성 조판으로 구성된 한 권의 시집.

## 구성
- `index.html` — 마크업
- `css/style.css` — 디자인 (라이트 톤 · 플립북 · 표지/면지/시 조판)
- `js/main.js` — 플립 엔진 · 시 데이터 · SVG 일러스트 · fit-to-page

## 로컬에서 보기
빌드가 필요 없습니다. `index.html`을 브라우저로 열면 됩니다.
정적 서버로 보려면:

    npx serve

## 배포
순수 정적 사이트라 Vercel · Netlify · GitHub Pages 등에 그대로 배포됩니다.
